This demo is based on TRmorph, a Turkish morphological analyzer. For the latest version of the analyzer this document please refer to TRmorph home page.

The analyzer is distributed under GNU GPL (version 2 or later). (NOTE: The lexicon provided is based on wordlists from Zemberek [3], however, it has been heavily modified. Note that, Zemberek is distributed under a different free license, MDL.)
